leetcode(1025)-除数博弈

SùXiǎolěi 2020-07-24 每日一题LeetCode动态规划

题目

爱丽丝和鲍勃一起玩游戏，他们轮流行动。爱丽丝先手开局。
最初，黑板上有一个数字 N。在每个玩家的回合，玩家需要执行以下操作：

  • 选出任一 x，满足 0 < x < N 且 N % x == 0
  • 用 N - x 替换黑板上的数字 N

如果玩家无法执行这些操作，就会输掉游戏。
只有在爱丽丝在游戏中取得胜利时才返回 True，否则返回 false。假设两个玩家都以最佳状态参与游戏。
示例1：
输入：2
输出：true
解释：爱丽丝选择 1，鲍勃无法进行操作。
示例2：
输入：3
输出：false
解释： 爱丽丝选择 1，鲍勃也选择 1，然后爱丽丝无法进行操作。
提示：

  • 1 <= N <= 1000
// N=1：A败
// N=2：A拿1，A胜
// N=3：A拿1，N变成2，根据N=2结论，A败
// N=4：A拿1或2，如果A拿1（玩家以最好状态参与游戏，即使用最优解），N变成3，根据N=3结论，A胜
// N=5：A拿1，N变成4，根据N=4结论，A败
// ...

class Solution {
    public boolean divisorGame(int N) {
        return N % 2 == 0;
    }
}
